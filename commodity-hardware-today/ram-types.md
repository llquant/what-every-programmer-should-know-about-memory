# 2.1. RAM 的种类

这些年来，RAM 已经出现过许多不同类型，而且每种类型之间都有差异，有时差异还非常显著。那些较老的类型如今大概只对历史学家有意义，我们不会探究它们的细节。我们将聚焦于现代 RAM 类型；这里也只会触及表面，探讨内核或应用程序员能够通过性能特征观察到的一些细节。

第一个值得关注的问题，是为什么同一台机器中会有不同类型的 RAM。更具体地说，为什么既有静态 RAM（Static RAM，SRAM[^5]）又有动态 RAM（Dynamic RAM，DRAM）。前者速度快得多，而且提供相同的功能。那么，为什么一台机器里的 RAM 不全都使用 SRAM？答案如你所料：成本。SRAM 的生产成本和使用成本都比 DRAM 高得多。这两个成本因素都很重要，而且后者正变得越来越重要。为了理解这些差异，我们来看一下 SRAM 和 DRAM 中一个存储位的实现方式。

在本节其余部分，我们会讨论一些 RAM 实现的低层细节。我们会把细节层级控制在尽可能低的程度。为此，我们会在“逻辑层级”上讨论信号（signal），而不是采用硬件设计师所需的那种层级；后者对本文目的来说没有必要。

[^5]: 在其他上下文中，SRAM 也可能指“同步（synchronous）RAM”。
